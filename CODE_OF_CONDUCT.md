# 行为准则

为了让 TravelShare 的协作过程保持高效、友善和可持续，参与者默认遵守以下约定。

## 我们鼓励的行为

- 尊重不同背景、经验和表达方式的同学。
- 讨论问题时聚焦事实、代码和方案，而不是针对个人。
- 在提出修改意见时给出可执行的建议。
- 主动补充上下文，帮助后来者更快理解项目。

## 我们不接受的行为

- 人身攻击、讽刺、羞辱或带有歧视性的表达。
- 恶意争论、反复打断他人或故意制造协作障碍。
- 未经允许公开他人的隐私、账号信息或敏感配置。

## 处理原则

- 轻微问题优先通过提醒和澄清解决。
- 持续性或严重问题可由维护者暂停相关讨论或拒绝合并改动。
- 涉及敏感情况时，请通过私下渠道联系项目维护者沟通。

## 适用范围

本准则适用于仓库内的 Issue、Pull Request、代码审查、文档讨论以及与本项目相关的协作交流。
