# 社区建设

## Discussion 分类

推荐启用 GitHub Discussions 并设置分类：
- 📣 Announcements - 公告
- 💡 Ideas - 功能建议
- 🙏 Q&A - 问答
- 🙌 Show and tell - 展示

## 贡献者认可

### All Contributors

```markdown
## Contributors

感谢所有贡献者！

<!-- ALL-CONTRIBUTORS-LIST:START -->
<!-- ALL-CONTRIBUTORS-LIST:END -->

[![All Contributors](https://img.shields.io/github/all-contributors/USER/REPO)](CONTRIBUTORS.md)
```

### 配置

文件位置: `.all-contributorsrc`

```json
{
  "projectName": "repo-name",
  "projectOwner": "username",
  "repoType": "github",
  "repoHost": "https://github.com",
  "files": ["README.md"],
  "imageSize": 100,
  "commit": true,
  "commitConvention": "angular"
}
```

## 赞助支持

文件位置: `.github/FUNDING.yml`

```yaml
github: [username]
patreon: username
open_collective: project-name
ko_fi: username
custom: ["https://example.com/donate"]
```

## 社区活动

### 定期活动
- 每周/每月社区会议
- 代码审查日
- 文档贡献日
- 新手友好 Issue

### 标签管理
- `good first issue` - 新手友好
- `help wanted` - 需要帮助
- `documentation` - 文档相关
- `bug` - Bug 报告
- `enhancement` - 功能增强
